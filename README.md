## 🚀 Хурдан эхлэх

```bash
# 1. Хамаарал суулгах
npm install

# 2. Өгөгдлийн сан бэлдэх (Prisma client + SQLite + seed контент)
npm run setup

# 3. Сервер асаах
npm run dev
```

Дараа нь хөтөч дээр **http://localhost:3000** нээнэ.

> 💡 `.env` файл нь тест default утгуудтай бэлэн ирдэг тул нэмэлт тохиргоо
> шаардлагагүй. Шууд ажиллана.

### 🔑 Тест эрхүүд

| Үүрэг | Нэвтрэх |
| --- | --- |
| **Админ** | `99000000` / `admin123` (Ажилтан таб) |
| **Багш** | `99001111` / `teacher123` (Ажилтан таб) |
| **Суралцагч** | Бүртгэл хуудаснаас **"Тест хэрэглэгчээр нэвтрэх"** товч дарна |

> Тест горимд (`TEST_MODE=true`) SMS OTP код дэлгэц дээр шууд харагдана.

## 🧪 Туршилт хийх (бүрэн урсгал)

1. `/register` → **"Тест хэрэглэгчээр нэвтрэх"** дарж шууд орно
2. **Сургалт** → курс сонгож **Элсэх** → QPay модал дээр **"Төлбөр баталгаажуулах (симуляци)"**
3. **Сургалт** → 4 модулийн агуулгыг үзэж тус бүрийг **Дуусгасан**
4. **Шалгалт** → 30 асуултад хариулж тэнцэх (80%+)
5. **Практик дадлага** → жолооны курс сонгож захиалах → **"Бүгдийг тэнцүүлэх"** (тест баталгаажуулалт)
6. **Гэрчилгээ** → дижитал сертификат + QR харагдана
7. **Профайл** → түрээсийн апп холбож урамшууллын оноо авах

Эсвэл автомат тестээр бүх урсгалыг шалгана (сервер асаалттай байх үед):

```bash
npx tsx scripts/e2e-test.ts
```

## 🏗️ Технологийн стек

- **Next.js 14** (App Router) + **TypeScript** + **Tailwind CSS**
- **Prisma ORM** + **SQLite** (тестэд тэг тохиргоо; production-д PostgreSQL руу солих боломжтой)
- **jose** (JWT, httpOnly cookie нэвтрэлт), **bcryptjs** (нууц үг)
- **qrcode** (QR код), **zod** (баталгаажуулалт), **lucide-react** (icon)

> Судалгааны баримт нь микросервис архитектур заасан ч, бүрэн ажиллагаатай,
> хялбар тест хийх боломжтой байхын тулд **нэг Next.js full-stack апп**
> (модуляр monolith) хэлбэрээр хэрэгжүүлсэн. Сервисүүд (Auth, Course, Exam,
> Practice, Certificate, Payment, Partner) нь API модулиуд болсон.

## 🔌 Гадаад нэгтгэлүүд (тест ↔ production)

### Хур (XYP) — нас баталгаажуулалт
- **Тест:** регистрийн дугаараас мэдээлэл (нэр, төрсөн огноо) гарган мок хариу буцаана.
- **Production:** `src/lib/xyp.ts` доtorх `fetchFromXyp()`-д бодит XYP REST/SOAP сервисийг
  (`WS100101_getCitizenIDCardInfo`) дуудна. `.env.local`-д `XYP_ACCESS_TOKEN` тохируулна.

### QPay — төлбөр
- **Тест:** мок invoice үүсгэж, "симуляци" товчоор төлбөрийг PAID болгоно.
- **Production:** `src/lib/qpay.ts` нь QPay v2 API (auth/token + invoice) дуудна.
  `.env.local`-д `QPAY_USERNAME`, `QPAY_PASSWORD`, `QPAY_INVOICE_CODE` тохируулна.
  Webhook: `POST /api/payments/webhook/qpay`.

### Хамтрагч / Түрээсийн апп API
- `POST /api/partner/verify-cert` — сертификат шалгах (Header: `X-API-Key`)
- `POST /api/partner/reward/grant` — урамшуулал өгөх
- `GET  /api/partner/stats` — статистик
- API key-г админ → Хамтрагчид хэсгээс авна.

## 🌐 Production-д шилжүүлэх

1. `.env.local` үүсгэж бодит нууц утгуудыг оруулна (`TEST_MODE=false`).
2. PostgreSQL руу шилжих: `prisma/schema.prisma` доtор `provider = "postgresql"`,
   `DATABASE_URL`-г Postgres холболт болгоно. `npx prisma migrate deploy`.
3. SMS gateway (Mobicom/Unitel), QPay, XYP-ийн бодит түлхүүрүүдийг холбоно.
4. `npm run build && npm start`.
